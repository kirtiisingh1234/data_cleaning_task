# data_cleaning_task
data cleaning using python(netflix dataset)


## Objective
Clean and preprocess Netflix dataset using Python.

## Steps Performed
- Handled missing values using fillna()
- Removed duplicates using drop_duplicates()
- Standardized text values
- Converted date format using to_datetime()
- Cleaned duration column (split into number and type)
- Renamed columns

## Tools Used
- Python
- Pandas

## Output
Cleaned dataset ready for analysis.
